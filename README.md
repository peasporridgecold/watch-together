# 같이보기 🍿

유튜브 영상을 친구와 **실시간으로 같이 보는** 웹앱입니다.
재생 · 일시정지 · 구간 이동이 서로에게 즉시 동기화돼요. 통화하면서 같이 보기 좋아요 📞

**바로 사용:** https://peasporridgecold.github.io/watch-together/

## 사용법
1. 한 명이 **방 만들기** → 초대 링크를 카톡 등으로 전송
2. 친구가 링크를 열면 자동으로 같은 방에 입장
3. 아무나 영상을 검색(또는 유튜브 링크 붙여넣기)해서 틀면 모두의 화면에서 같이 재생

## 동작 방식
- **음성 대화 내장** — 전화 앱 통화는 OS가 유튜브 소리를 막으므로, 앱 안에서 WebRTC로 직접 통화 (영상 소리와 목소리가 같이 들림)
- 서버 없이 브라우저끼리 직접 연결 (WebRTC / PeerJS)
- 방장이 3초마다 현재 재생 상태를 방송해 어긋남을 자동 보정
- 모바일 자동재생 정책 대응: 게스트는 처음 한 번 화면을 탭하면 따라가기 시작
